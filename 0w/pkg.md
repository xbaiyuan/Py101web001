
# 包的凌乱

自从 windows 转到类 unix 系统来以后，各种包及管理工具有时给我造成困扰

现在来梳理一下包管理

尝试回答下面几个问题

- 哪些包自动包含 Python 
- 哪些包自动包含 Pip
